## 🌐 About This Project

> **Scraping Domination** is a collection of powerful Python web scrapers that extract real-world data from live websites. From pharmaceutical products to cryptocurrency proxies, stock markets to books — this repo covers it all.

The goal is simple: **dominate the web with clean, efficient data extraction scripts** using Python's most battle-tested libraries.

<img src="https://user-images.githubusercontent.com/73097560/115834477-dbab4500-a447-11eb-908a-139a6edaec5c.gif" width="100%">

## 🕷️ Scrapers Overview

<div align="center">

| 📁 File | 🎯 Target | 🛠️ Method | 📤 Output |
|:---|:---|:---|:---|
| `scraping.py` | 💊 PharmEasy — Medicine Products | API + `requests` + `json` | CSV |
| `crypto_data.py` | 🔐 Proxy Lists + 💰 Yahoo Finance Crypto | `requests` + `BeautifulSoup` + `re` | CSV |

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Yuvrajsinghko/Scraping-Domination.git
cd Scraping-Domination
```

### 2. Install Dependencies

```bash
pip install requests beautifulsoup4
```

### 3. Run a Scraper

```bash
# Scrape PharmEasy medicine data
python scraping.py

# Extract crypto proxies
python crypto_data.py

# Scrape book titles
python books_name.py

# Pull stock market data
python stocks_data.py
```

> 📁 Output CSVs are saved in the `scraped_data/` directory.

## 💡 Key Highlights

- 🎯 **Real-World Targets** — Scrapes live production websites (PharmEasy, Yahoo Finance, and more)
- ⚡ **API-First Approach** — `scraping.py` reverse-engineers the PharmEasy REST API for fast, clean data
- 🔄 **Regex-Powered Extraction** — `crypto_data.py` uses regex patterns to pull proxy IPs from raw HTML
- 📤 **CSV Output** — All scrapers write structured, ready-to-analyze data to CSV files
- 🧩 **Modular Design** — Each scraper is a standalone script, easy to run and extend
